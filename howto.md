
# КАК СОЗДАТЬ КОНФИГ:
1) Создайте.
2) Готово.
## Пример
```yml
race_id: 
  race: true/false
  type: "строка"
  class: "строка"
  craft: "строка"
  attributes:
    MAX_HEALTH: 20
  effects:
    SPEED: 1
  foods:
    - rotten_flesh
    - dirt
  sub_attributes:
    MAX_HEALTH: 10
  sub_effects:
    REGENERATION: 1
  spec:
    armor: "IRON"
  menu_head: "строка"
  menu_name: "строка"
  menu_lore:
    - "строка1"
    - "строка2"
  height: 85
  biomes:
    - "end_midlands"

  skin:
    f: "строка"
    m: "строка"
```
### Описание полей
**race_id** - внутреннее ID расы, только маленькие английские буквы и цифры и "_", без пробелов, можно оставить любое\
**race** - true если это раса, false если листочек с информацией (тогда type = "info")\
**type** - одно из 3-ех вариантов:
1) "info" - для листочков с информацией в меню
2) "race" - раса, выбираемая из меню
3) "combined" - раса, созданная ядром перерождения.\

Дальнейшие поля применимы только к type = "race" или "combined"

**class** - класс расы, например человеко-подобные или зверолюды:
- "humanlike"
- "animaLike"
- "other"
- "combined"\
**slot**: 0 - в каком слоте будет эта раса, только если type = 'race'.\
**craft**: - крафт расы, только для type = 'combined'.
- Крафт создается из 2 рас, и имеет такой общий вид: "класс;раса+класс;раса"
- Тут, класс это "class", одно из выше-указанных полей, классы расс нужных для создания,
- А "раса" это уточнение расы нужной для создания, или *, если применима любая раса из текущего класса.
- Например "humanlike;dwarf+animaLike;*" - для создания требуется класс humanlike (человеко-подобные), и конкретно человек,
и вторая раса должна быть из класса зверолюды, но конкретная раса не важна (т.е. может быть как каджит, так и дракончик) 

**attributes**: Список из артибутов которые должны быть у расы в "обычное" время. См [Attributes](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/attribute/Attribute.html).\
Каждое из полей ниже должно быть СТРОГО одним из атрибутов с сайта выше, со значением этого атрибута у расы. Стандартные значения смотреть в файле /src/main/java/org/pythonchik/trueyandere/TrueYandere.java, переменная AttributeDefaults.\
**GRAVITY**: 0.09 - пример с атрибутом гравитации. Тут чуть повышена (с 0.08)

**effects**: Список эффектов у расы в "обычное время". См [Effects](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/potion/PotionEffectType.html).\
Каждое из полей ниже должно быть СТРОГО одним из эффектов с сайта выше, со значением - уровнем силы эффекта.\
**REGENERATION**: 1 - пример эффект регенерации 1-ого уровня.
